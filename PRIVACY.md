# Privacy Policy for NetStat

**Last updated: January 12, 2026**

## Overview

NetStat ("the App") is a network speed monitoring application developed by MatanelP. This privacy policy explains how the App handles user data.

## Data Collection

**NetStat does not collect, store, or transmit any personal data.**

The App only accesses:
- **Network traffic statistics**: Used locally on your device to calculate and display current download/upload speeds. This data is never stored or transmitted.
- **Device boot events** (optional): Only if you enable "Start on boot" feature, to automatically start monitoring.

## Data Storage

All settings and preferences are stored locally on your device using Android's SharedPreferences. No data is sent to external servers.

## Third-Party Services

NetStat does not use any third-party analytics, advertising, or tracking services.

## Permissions

The App requests the following permissions:
- **Notification permission**: To display network speed in the status bar
- **Network state access**: To read network traffic statistics
- **Boot completed** (optional): To start monitoring on device boot

## Children's Privacy

NetStat does not knowingly collect any information from children under 13 years of age.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
